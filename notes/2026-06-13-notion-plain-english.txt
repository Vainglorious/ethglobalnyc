THE PUSH SCRIPT
  - notion/push_to_notion.py — pure-stdlib (+certifi) Markdown -> Notion blocks converter.
    Usage: python3 notion/push_to_notion.py <PARENT_PAGE_ID_or_URL>
    Reads token from notion/.env. Handles headings, bullet/numbered lists, blockquotes,
    code fences, TABLES, dividers, and inline **bold** / `code` / *italic*. Creates one
    subpage per doc, appending blocks in chunks of <=90 (Notion's 100-block limit).
  - SSL gotcha: python.org Python 3.13 on macOS ships without CA certs, so urllib threw
    CERTIFICATE_VERIFY_FAILED (curl worked because it uses the system trust store). Fixed
    by building an ssl context from certifi.where().
