HONEST CAVEATS (asked + answered)
  - ARC / USDC: NO real settlement enforced. verified="stub" -> ANY non-empty
    X-PAYMENT passes. The 402->pay->200 HANDSHAKE is real; enforced Arc/USDC payment
    is NOT (documented TODO). No restriction beyond sending the header.
  - "Officially closed?" two layers:
      * markets dataset = definitive per-market verdict (closed + price 0/1). This is
        how we know Scotland won / others lost / France open.
      * uma_events dataset = the on-chain resolution ACTIVITY (Propose->Settle) behind
        those closures. Mapping a specific Settle event -> a specific one of OUR markets
        needs decoding the questionID from the event topics — NOT yet wired (TODO). So
        UMA proves resolution activity; the markets data gives the per-market answer.
