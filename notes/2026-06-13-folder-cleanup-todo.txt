NOTES / GOTCHAS
--------------------------------------------------------------------------------
- Use `git mv` (not plain mv) so history follows the files.

  agentkit). When moving the standalone spike folders, don't delete the
  integrated versions inside colony.

  not break those relative paths (frontend reads public/data/*.jsonl, not colony
  paths directly, so this should be safe — verify).
- Secrets check before any public push: .env, secrets/*.local.json, pg.session,
  agent-wallets stores. Several folders carry live keys (e.g. dynamic/.env).
